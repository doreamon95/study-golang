## 基础与环境

首先，官方入门（  [官方网址](https://tour.golang.org)  [中文网址](http://go-tour-zh.appspot.com)），了解语法，没搭环境在网站边看边学基础。

如果想搭建环境跟着操作，可以在[官方推荐工具](https://github.com/golang/go/wiki/IDEsAndTextEditorPlugins)中挑选，个人推荐vscode（原因主要是：可能会涉及多种语言开发，不想安装太多软件）。

学习时候如果想查找资料可以去[golang社区](https://studygolang.com)瞅瞅，或者时间充裕可以从[无闻golang入门视频](https://www.bilibili.com/video/av11069727?from=search&seid=11021125346744784199)开始学习起走。当然，若是时间不充裕又想大概概览一下golang，那么推荐[这个视频](https://www.youtube.com/watch?v=SqrbIlUwR0U&t=2127s)。

## Web框架

golang中web框架也有很多，主要列举以下三个：

- [gin](https://gin-gonic.github.io/gin)：Go语言编写的HTTP Web框架
- [beego](https://beego.me)：面向Go编程语言的开源高性能web框架，
- [Iris](https://iris-go.com)：号称最快的Go语言Web框架 完备MVC支持

ORM框架：

- [xorm](https://godoc.org/github.com/go-xorm/xorm)
- [gorm](https://gorm.io/docs/)

RPC框架：

- [grpc](https://grpc.io/docs/guides/)
